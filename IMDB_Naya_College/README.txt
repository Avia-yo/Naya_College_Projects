This project focuses on IMDB dataset.

The project includes the following main steps:

Data Import: The IMDB dataset is imported using the Kaggle API or from a CSV file.

Data Cleaning: Null values and unnecessary columns are removed from the dataset to ensure data quality.

Data Manipulation: A utils class is implemented with methods to sort data, find top values, and perform calculations such as sum, average, and maximum values.

Data Saving: The processed data is saved to a CSV files and json file.

Data Visualization: Visualizations such as plots are created to visualize the correlation between different features in the dataset
